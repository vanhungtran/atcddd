# Convert a numeric value between compatible units

Convert a numeric value between compatible units

## Usage

``` r
.ddd_convert(value, from_unit, to_unit)
```

## Arguments

- value:

  Numeric value to convert.

- from_unit:

  Character; source unit.

- to_unit:

  Character; target unit.

## Value

Converted numeric value. Returns `NA` if the units are incompatible or
unrecognised.
